# 🚀 GitHub Profile Analyzer

A Python application that analyzes GitHub users through the GitHub REST API and generates detailed reports, charts and statistics.

---

## Features

✅ GitHub REST API

✅ Terminal UI

✅ Language Statistics

✅ Repository Analysis

✅ PNG Charts

✅ PDF Reports

✅ JSON Export

---

## Technologies

Python

Requests

Rich

Matplotlib

FPDF

GitHub REST API
